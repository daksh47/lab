HOW TO SAVE PHP FILES

OPEN XAMPP
	- GO TO SEARCH and type XAMPP

START THE APACHE SERVER

GO TO THIS PC

THEN OPEN WINDOWS C

THEN SCROLL DOWN AND OPEN xampp

THEN OPEN htdocs

THEN CREATE A NEW FOLDER NAMED "prg"

THEN OPEN prg

SAVE ALL PHP FILES HERE


HOW TO EXECUTE PHP FILES

OPEN ANY WEB-BROWSER

IN THE URL TYPE "localhost/prg/prg8.php"

REPLACE "prg8" with your php file name

HIT ENTER
